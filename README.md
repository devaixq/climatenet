# ClimateNet

## 프로젝트 개요
ClimateNet은 위성 기반 기후 제어 시스템을 목표로 하는 프로젝트이다.  
본 시스템은 위성 데이터를 활용하여 전 세계 기후 변화를 모니터링하고 이상 기후를 예측한다.

또한 미세먼지와 같은 대기 오염 문제를 분석하고, 인공 강우 등 기후 제어 기술을 활용하여 환경 문제 대응을 지원한다.

## 프로젝트 목표
- 위성 기반 기후 데이터 수집: 전 세계 기후 정보를 실시간으로 수집
- 기후 변화 분석 및 예측: 이상 기후 및 재난 상황 예측
- 대기 오염 및 미세먼지 모니터링: 지역별 환경 문제 분석
- 기후 제어 기술 지원: 인공 강우 등 대응 전략 분석 및 시뮬레이션
- 글로벌 기후 데이터 공유: 연구 기관 및 국제 기구와 통합 데이터 활용
